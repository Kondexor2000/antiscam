# Laboratorium 1: wykrywanie ryzyka phishingu

## Cel

Student potrafi uruchomic API i zinterpretowac wynik oceny ryzyka.

## Zadania

1. Uruchom Python API.
2. Wyslij wiadomosc neutralna i zapisz wynik.
3. Wyslij wiadomosc z kodem BLIK i pilnym kontekstem.
4. Porownaj `risk_score`, `status` i `reasons`.

## Kryteria zaliczenia

- 40% poprawne uruchomienie API,
- 30% poprawne wykonanie dwoch zapytan,
- 30% interpretacja wyniku.
