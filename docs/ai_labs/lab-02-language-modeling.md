# AI Lab 2: language modeling and text representation

## Goal

Demonstrate NLP preprocessing, vector representation and an n-gram language model.

## Tasks

1. Tokenize two Polish messages.
2. Build bag-of-words vectors.
3. Compute cosine similarity.
4. Train `NGramLanguageModel` and query a token probability.

## Assessment

- 25% tokenization,
- 25% vector representation,
- 25% similarity interpretation,
- 25% n-gram probability explanation.
