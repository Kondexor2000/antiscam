# AI Lab 3: CAT, terminology and named entities

## Goal

Show how AntiScam can support translator workflows and machine-translation preparation.

## Tasks

1. Extract terms from a phishing education text.
2. Extract named entities from a message containing an organization name.
3. Add two entries to `TranslationMemory`.
4. Request a suggestion for a similar source segment.

## Assessment

- 30% terminology extraction,
- 30% named entity extraction,
- 30% translation-memory suggestion,
- 10% discussion of MT evaluation limits.
