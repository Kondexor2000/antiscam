# AI Lab 4: dialogue, empathy and knowledge engineering

## Goal

Build a small task-oriented interaction around phishing support.

## Tasks

1. Use `AntiScamDialogBot` with a BLIK message.
2. Inspect returned intent, message and emotion.
3. Add mitigation facts to `KnowledgeGraph`.
4. Query facts by relation.

## Assessment

- 30% chatbot response,
- 25% emotion label explanation,
- 25% knowledge graph facts,
- 20% ethics and limitations.
