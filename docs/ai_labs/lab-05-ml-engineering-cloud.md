# AI Lab 5: ML engineering and cloud deployment

## Goal

Connect the AI helpers with engineering practices and cloud deployment models.

## Tasks

1. Run `python -m pytest tests/unit/test_ai.py`.
2. Explain how the tests work as regression checks for ML helpers.
3. Inspect `cloud_deployment_profile`.
4. Choose IaaS, PaaS, FaaS or SaaS for a classroom AntiScam deployment and justify the choice.

## Assessment

- 30% successful tests,
- 25% regression-testing explanation,
- 25% cloud model selection,
- 20% deployment-risk discussion.
